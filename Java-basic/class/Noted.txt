sebuah class dapat memiliki jenis variabel:

variabel lokal:variabel yg didefinisikan  di dlm sebuah konstruktor atau blok.
variabel ini dideklarasikan dan diinisialisasi di dlm method dan kemudian akan dihapus saat method selesai dijalankan.


variabel instance:variabel  yg ada di dalam class tetapi berada di luar method.
variabel ini juga dapat diakses dari dalam method,konstruktor,atau blok dari kelas tersebut

variabel class:variabel yg dideklarasikan di dalam sebuah class filuar method,dsn menggunakan kata kunci static.
